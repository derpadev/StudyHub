# StudyHub

A Reddit-style full-stack social media platform for students to discuss classes, majors, and other campus-related topics. Ideal for UCI students, with potential to expand to other colleges in the future.

## Features
- Users can **create posts** and **communities**  
- **Like** and **comment** on posts  
- **Real-time updates** for posts, likes, and comments  
- **Login with GitHub** for authentication  

## Tech Stack
- **Frontend:** React, Tailwind CSS  
- **Backend:** Supabase  
- **Additional Libraries:**  
  - @supabase/supabase-js  
  - @tanstack/react-query  
  - react-router
